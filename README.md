# Desktop-Cleaner

A lightweight Python automation script for **Linux Mint XFCE**. It keeps your workplace tidy by scanning your desktop and moving loose files to dedicated folder.

## Features
   **RAM Optimized:** Designed for smooth performance on 4GB RAM systems.
   **Automated Sorting:** Moves files from `~/Desktop` to `~/Documents/organized`.
   **Safety First:** Automatically ignores the `cleaner.py` script so it doesn't move itself.

## Usage
  Open your terminal.
  Run the script:
  ```bash
python3 cleaner.py
